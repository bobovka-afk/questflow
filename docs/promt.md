# Questflow — промты для генерации графики

**Один абзац = один промт = один файл.** Копируй **весь абзац** под строкой «Файл» целиком в генератор — в нём уже есть стиль (пиксель-арт + фэнтези RPG), размер, фон, композиция и запреты. Таблица ниже — только правила именования и анти-дублей.

### Правила без дублей

| Правило | Пример |
|---------|--------|
| Закрытый сундук = только `0.png` | Нет `chest_rare.png`; `rare/0.png` = иконка + кадр tap |
| Один файл — разный масштаб в CSS | `dust/dust.png`; не делать `dust@2x.png` без отдельного pipeline |
| Один ключ БД = один PNG | `frame_epic_flame` → тот же `mystic.png` (см. `cosmetics.ts`) |
| Locked achievements | Только **CSS grayscale**, не `{key}_locked.png` |
| Перерисовка placeholder | `ui/checkin/check.png` **на месте**, не второй `flame.png` |
| Босс ≠ сундук босса | `bosses/nasadka.png` (существо) ≠ `chests/boss/nasadka/0.png` |
| Milestone 7/30 = achievement | `achievements/streak_7.png` и `streak_30.png` также для модалки порогов серии |
| Milestone 14 | Только `achievements/streak_14.png` (отдельной achievement в сиде нет) |

**Уже в проекте (промт не нужен):** рамки `bronze/silver/gold/mystic`, фоны профиля (4 шт.), `chests/common/0–4.png`, `chests/rare/1–5.png`, `dust/dust.png`, `ui/xp/xp.png`, `ui/checkin/check.png`, intro, clear-портреты (12), placeholder ачивок `color.png` / `black.png`.
