# Questflow — промты для генерации графики

**Один абзац = один промт = один файл.** Копируй **весь абзац** под строкой «Файл» целиком в генератор — в нём уже есть стиль (пиксель-арт + фэнтези RPG), размер, **явные требования к прозрачному фону (PNG-32 / RGBA)**, композиция и запреты. Таблица ниже — только правила именования и анти-дублей.

### Правила без дублей

| Правило | Пример |
|---------|--------|
| Закрытый сундук = только `0.png` | Нет `chest_rare.png`; `rare/0.png` = иконка + кадр tap |
| Один файл — разный масштаб в CSS | `dust/dust.png`; не делать `dust@2x.png` без отдельного pipeline |
| Один ключ БД = один PNG | `frame_epic_flame` → тот же `mystic.png` (см. `cosmetics.ts`) |
| Locked achievements | Только **CSS grayscale**, не `{key}_locked.png` |
| Перерисовка placeholder | `ui/checkin/check.png` **на месте**, не второй `flame.png` |
| Босс ≠ сундук босса | `bosses/rust_king.png` (существо) ≠ `chests/boss/rust_king/0.png` |
| Milestone 7/30 = achievement | `achievements/streak_7.png` и `streak_30.png` также для модалки порогов серии |
| Milestone 14 | Только `achievements/streak_14.png` (отдельной achievement в сиде нет) |

### Прозрачность фона (иконки, сундуки, портреты, медали)

Для всех ассетов с прозрачностью в промте **обязательно** дублируются явные требования к экспорту:

| Требование | Значение |
|------------|----------|
| Формат | **PNG-32**, **RGBA**, настоящий альфа-канал |
| Вне объекта | **alpha=0** (полностью прозрачно) |
| Запрещено | белый/чёрный/серый фон, solid backdrop, «шахматная» сетка прозрачности, fake transparency grid, RGB/JPEG без альфы |
| Экспорт | `transparent background only`, isolated sprite, empty canvas вне силуэта |
| Fallback | если модель не даёт альфу — chroma key `#00FF00`, но предпочтительно сразу PNG с прозрачностью |

**Исключения (фон НЕ прозрачный):** `cosmetics/backgrounds/*`, широкие баннеры UI — там явно указано «непрозрачный на весь холст».

**Уже в проекте (промт не нужен):** рамки `bronze/silver/gold/mystic`, фоны профиля (4 шт.), `chests/common/0–4.png`, `chests/rare/1–5.png`, `ui/dust/dust.png`, `ui/xp/xp.png`, `ui/check/check.png`, `ui/health/health.png`, `ui/level/level.png`, `ui/mana/mana.png`, intro, clear-портреты (12), `ui/achievments/*` + `universal.png`.

## Чеклист после генерации

- [ ] **Прозрачность:** PNG-32 / RGBA; вне объекта alpha=0; нет белого фона, checkerboard и RGB без альфы (проверить в просмотрщике с шахматкой).
- [ ] Имя файла = `CosmeticItem.key` или путь из таблиц выше.
- [ ] Сундуки различимы в **32 px** (Common / Rare / Epic / Boss).
- [ ] Рамки: прозрачный центр проверен на тестовом портрете 1024×1024.
- [ ] Фоны: центральный квадрат читается в crop **300×300** с `object-fit: cover`.
- [ ] Boss-иконки согласованы с boss-сундуками и raid-рамками по палитре.
