Начисление XP за задачу — 10 вариантов

XP-1 · Текущий (baseline)

как в prod

Вертикальный стек: иконка XP → число → подпись «За закрытие карточки» → HP. Градиент primary, скругление 20px. Классы: trello-reward-toast--task.

TaskRewardToast
XP 50 XP

За закрытие карточки

Здоровье +5

XP-2 · Горизонтальная капсула

компактный

Одна строка: XP слева, текст справа. Меньше высота — не перекрывает доску. Белый фон + синяя обводка, хорошо читается на pixel-фоне.

компактный chip
XP +50
За закрытие карточки
+5 HP

XP-3 · Pixel RPG badge

рекомендуем

В духе Questflow/pixel-shell: жёсткая рамка, тень-смещение, кремовый блок с крупным числом. Отделяет «награду» от «причины». HP — вторичной строкой зелёным.

RPG loot popup
+50 XP
Карточка закрыта ♥ +5 HP

XP-4 · Split card

акцент на числе

Крупное число XP слева (hero), описание и HP-chip справа. Светлая карточка на тёмной доске — контраст без агрессивного градиента.

hero number
50 XP
За закрытие карточки
Задача выполнена
+5 HP

XP-5 · Minimal float

минимализм

Только «+50 XP» без подписи — HP показывать отдельным микро-тостом или в профиле. Быстрое «поп-ап» число с анимацией всплытия (Duolingo-style).

floating number
+50XP

XP Hero — 5 вариантов в стиле C-3 Flame hero

XP-6 · Star hero

как C-3

Прямой аналог C-3 для XP: тёмный фон, золотое сияние, крупная ⭐ по центру. Главная строка — причина («За закрытие карточки»), XP и HP — вторичным текстом.

star celebration
⭐

Холодное голубое свечение и кольцо XP — ощущение мгновенной награды за действие. Читается на доске без лишнего декора.

Зелёное свечение и ♥ — когда важно подчеркнуть восстановление здоровья вместе с XP. Подходит для привычек и личных дел.

milestone

Фиолетово-золотой градиент и 💎 — «драгоценная» награда. Хорошо для крупных XP (100) и редких событий (порог серии, квест).

Чекин / серия за день — 5 вариантов

C-1 · Текущий (baseline)

как в prod

Оранжево-красный градиент, XP за серию сверху, подпись «Серия» + иконка, блок серии снизу. Классы: trello-reward-toast--checkin, CheckinStreakCounter.

CheckinRewardToast
XP 100 XP

🔥 Серия

🔥 7 дней · Серия

C-2 · Двухслойный (серия + XP)

рекомендуем

Главный акцент — цифра серии (крупный баннер), XP — отдельным chip снизу. Меньше «монолитного» оранжевого блока; проще читать при серии 7→8 с анимацией.

streak first
🔥 7 дней
подряд
XP +100 XP за день

C-3 · Flame hero

эмоциональный

Большое пламя по центру, тёмный фон — «мilestone moment». XP вторично мелким текстом. Хорошо для порогов серии (7, 14, 30 дней).

celebration
🔥
7-й день подряд
Серия засчитана · +100 XP

C-4 · Календарный чип

без слова «чекин»

Мини-календарь с числом серии вместо flame-pill. Текст «Активный день» / «Серия сохранена» — нейтральная продуктовая формулировка (как в квестах «активные дни»).

calendar chip

C-5 · Glass overlay

информативный

Frosted glass: иконка + заголовок, прогресс до следующего порога серии (например 7/14). Подходит, если хотите в тосте намекать на milestone без отдельного модала.

Краткий итог

  • XP: для pixel-стиля Questflow — XP-3 (RPG badge); для dark hero как C-3 — XP-6 (Star hero) или XP-10 (Combo) если нужны HP/мана.
  • Чекин: для эмоционального момента — C-3 (Flame hero); для читаемости серии и XP — C-2 (двухслойный).
  • Prod сейчас: XP-1 + C-1 — можно оставить и точечно улучшить типографику/отступы.
