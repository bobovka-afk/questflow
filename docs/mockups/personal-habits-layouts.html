Раздел solo-привычек — 10 вариантов вёрстки

Референс: Habitica (Привычки / Ежедневные задачи / Задачи). Маршрут в roadmap: /personal. Откройте файл в браузере и переключайтесь по якорям.

Варианты названия вкладки в левом rail

Личное Охватывает все три колонки. Нейтрально, не вводит в заблуждение. Совпадает с roadmap. Иконка: дом / звезда · flyout: «Личное»
Привычки Узнаваемо для фанатов Habitica, но внутри ещё ежедневные задачи и задачи — может сбивать. Иконка: ± · flyout: «Привычки»
Рутина Акцент на повторяющихся делах (привычки + ежедневные задачи). Задачи — «разовые» — чуть в стороне. Иконка: цикл / календарь
Дела Коротко, понятно соло-пользователю. Менее «игровое», зато универсально. Иконка: список / галочка
Журнал RPG-оттенок: «дневник героя». Хорошо стыкуется с квестами и XP. Иконка: свиток / книга
Тренировка Геймификация: «прокачка навыков». Сильный тон, не всем зайдёт. Иконка: меч / щит
Ежедневная задача Упор на dailies; привычки и todo вторичны в названии. Иконка: солнце / календарь
Мои задачи Близко к todo-листу. Привычки с +/- могут казаться «не задачами». Иконка: чеклист
Путь Короткий RPG-бренд («путь героя»). Загадочно без подсказки flyout. Flyout: «Личный путь»
Solo Технически точно (solo-режим), но английский в русском UI. Иконка: одиночная фигура
