/workspaces — рабочие пространства

5 вариантов оформления в актуальном дизайне Questflow: icon rail, parchment-тема, modern UI (скругления, glass, мягкие тени). Демо-данные: 3 пространства.

01 Split 02 Плитки 03 Таблица 04 Альбомы 05 Аккордеон
01

Split master / detail (как сейчас)

Список слева, детали справа. Двойной клик по пункту — переход к доскам. Знакомый паттерн, но на узких экранах колонки складываются.

✓ Уже в коде · ✓ Быстрый обзор без перехода · ✗ Много «пустого» места справа при одном WS

02

Сетка плиток (Trello-style)

Каждое пространство — цветная плитка с названием и краткой статистикой. Клик открывает доски. Плитка «+» для создания — как на пустом экране досок.

✓ Визуально богато · ✓ Масштабируется на много WS · ✓ Сразу видно «сколько всего»

Таблица с быстрыми действиями

04

Карточки-альбомы с цветной шапкой

Вертикальный стек широких карточек: цветная полоса-заголовок, описание и действия в теле. Хорошо читается на десктопе, каждое WS — отдельный «блок проекта».

✓ Чёткая иерархия · ✓ Место для описания · ✓ RPG-ощущение «локаций»

05

Компактный аккордеон

Узкая лента карточек: аватар-инициал, название, роль. Раскрытие показывает описание, статы и кнопки — без второй колонки. Отлично на мобильных и при 3–5 WS.

✓ Экономит ширину · ✓ Все действия в одном месте · ✓ Легко добавить drag-sort
