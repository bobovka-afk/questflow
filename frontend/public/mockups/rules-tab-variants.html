Персонаж

Вариант 1 — Карточки по темам

Каждое правило — отдельная панель вместо длинного списка.

Задания? Правила

Один персонаж

Прогресс учитывает активность во всех workspace. Без персонажа награды не начисляются.

Игровые сутки

Лимиты и серия — по Europe/Moscow. Сброс XP за карточки в полночь пояса.

Опыт за карточки

+10 XP за первое закрытие. Исполнитель или закрывший. Не более 5 раз в сутки.

HP и серия

Макс. 100 HP. Чекин при первом XP за день. Штраф −5 HP без активности вчера.

Квесты и сундуки

Дневные/недельные задания → сундук. Косметика без статов. Дубликат → пыль.

Вариант 2 — Свиток правил

Классический пергамент, как «книга заклинаний».

? Правила

☆ КАК ЭТО РАБОТАЕТ ☆

  • Один персонаж на аккаунт. Прогресс во всех workspace.
  • Игровые сутки. Часовой пояс сервера Europe/Moscow.
  • Опыт за карточки. +10 XP, лимит 5/день на персонажа.
  • Серия. Авто-чекин при первом XP за день, бонусы на 7/14/30 дней.
  • Квесты. Сундук за каждый выполненный квест.
  • Пыль. Дубликаты косметики → обмен на сундуки в магазине.

Вариант 3 — Аккордеон

Свёрнутые секции — удобно для длинного гайда.

Персонаж и сутки
Один персонаж. Игровой день по Europe/Moscow. Без персонажа — нет наград.
XP и лимиты
+10 XP за карточку, макс. 5/день. Повторное закрытие той же карточки — без XP.
HP и серия
Макс. 100 HP. Чекин +5 XP при первой активности. −5 HP за день без активности.
Квесты, сундуки, пыль
Квест → сундук. Дубликат косметики → пыль (15/40/80). Магазин: 50/120/250 пыли.
Достижения
Разовые награды за карточки, серию, уровень, коллекцию.
